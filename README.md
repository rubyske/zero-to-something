### 1. Definisi Inti

| Konsep | Penjelasan | Peran Anda |
| :--- | :--- | :--- |
| **Pemrograman** | Proses menulis instruksi logis yang dapat dieksekusi oleh komputer untuk menyelesaikan suatu masalah (Algoritma). | Anda adalah **Programmer** (Penulis instruksi). |
| **Bahasa Pemrograman** | Sintaks dan aturan yang digunakan sebagai jembatan komunikasi antara manusia dan mesin (misal: Java, JavaScript, Python). | Anda akan menggunakan **Java** dan **JavaScript** (Full-Stack) serta **Python** (AI). |

### 2. Version Control & Kolaborasi (Wajib)

| Alat | Deskripsi | Aksi Wajib |
| :--- | :--- | :--- |
| **Version Control (VC)** | Sistem untuk merekam perubahan pada *file* dari waktu ke waktu, memungkinkan kolaborasi tim dan kembali ke versi stabil. | Pahami pentingnya VC untuk kolaborasi profesional. |
| **Git** | Perangkat lunak Version Control terdistribusi yang dipasang secara lokal. | Kuasai perintah dasar: `init`, `add`, `commit`, `push`, `pull`, `branch`, `merge`. |
| **GitHub** | Platform *cloud* untuk *hosting* repositori Git. | Buat akun, pelajari `Pull Request`, dan jadikan sebagai **Portofolio Hidup** Anda. |
